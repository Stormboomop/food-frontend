# 📖 About

Boom's Kitchen is a full-stack food ordering web application designed to provide a smooth online food ordering experience. Users can browse the menu, add items to the cart, adjust quantities, and place orders through a responsive interface.

The application is powered by a Node.js backend with a MySQL database for managing users, food items, and orders.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Stormboomop/food-frontend.git
```

Clone Backend

```bash
git clone https://github.com/Stormboomop/food-backend.git
```

Install Dependencies

```bash
npm install
```

Create Environment Variables

```env
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
PORT=
```

Run Backend

```bash
npm start
```
