## рџ“– About

Boom's Kitchen is a full-stack web application independently developed during **May 2025** as a personal project. It demonstrates complete frontend and backend development using modern web technologies.

The project features a responsive user interface built with HTML, CSS, and JavaScript, while the backend is powered by Node.js and Express.js. SQLite is used to manage users, food items, and order information.

The application simulates a real-world online food ordering system by implementing authentication, cart management, checkout, and order storage.
